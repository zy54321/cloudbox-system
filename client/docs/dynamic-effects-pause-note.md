# 动态场景 runtime effects（unitPulse 等）— 专项暂停说明

**状态**：已暂停，与当前**稳定演示基线**解耦。本文仅作归档，不改变程序行为。

## 暂停原因（摘要）

针对 **unitPulse** 等配置驱动叠加效果曾出现或怀疑与主路径强耦合的问题，包括但不限于：

- 初始化阶段**相机聚焦**异常；
- **场景切换**后初始化异常；
- 关键点**叙事卡住**、或**飞机模型消失**等表现。

故在**未单独验证**前，**整体不再启用**依赖 `dynamic_scenarios.json` 中 runtime **effects** 的交付与维护路径。

## 当前策略

- **非专项**不得再向 **`dynamic_scenarios.json`** 添加或依赖 **runtime effects** 字段作为产品能力。
- **若未来重启**：必须先做 **独立 PoC 页面** 或 **独立分支** 专项验证，**不得**直接接入当前稳定**动态交互主页面**。
